A standard library do Go é bastante poderosa.

Portanto, não utilizar um framework é perfeitamente aceitável

Falar sobre a divisão de pacotes. Sobre a possibilidade de baixar apenas o necessário, mesmo da STD lib. * 
package main

import (
	"fmt"
	"io"
	"net/http"
)

func hello(w http.ResponseWriter, r *http.Request) {
	io.WriteString(w, "Hello, TDC!")
}

func main() {
	http.HandleFunc("/", hello)
	fmt.Println("Serving http...")
	http.ListenAndServe(":8000", nil)
}
