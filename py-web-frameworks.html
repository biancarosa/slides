PRO TIP

Só porque é simples de sair do "zero" não significa que caso você escolha levar uma aplicação pra produção, você não precisa entender como ele funciona "por baixo dos panos".

Flask

  • Servidor Web para Desenvolvimento
  • Framework de testes unitários
  • Roteamento
  • Jinja2 templates
  • Sessões

Quando faz sentido

  • se você acha que o Django vai ser um canhão
  • para aplicações menores e microsserviços
  • se você quer fugir de ORMs
  • quando você quer / precisa de mais liberdade

Projetos bacanas

  • https://github.com/kennethreitz/httpbin

PRO TIP

Especialmente se você está começando, a estrutura de pastas é completamente livre e requer certa senioridade até você conseguir criar uma que faz sentido do zero. No início, procure projetos em Flask e se espelhe neles.
